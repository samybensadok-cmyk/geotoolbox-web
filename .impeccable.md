# Design Context — GEO Toolbox

## Target Audience
SEO professionals, agency operators, and in-house marketers tracking AI/LLM visibility (ChatGPT, Perplexity, Gemini, Claude, Google AI Overviews). Technical buyers; skim-readers; comparison shoppers evaluating GEO tools.

## Use Cases
1. Researching what GEO Toolbox does before signing up
2. Reading blog content for thought-leadership trust signals
3. Entering the app (`/app/*`) to run AI visibility scans
4. Reference: Privacy/Terms for legal compliance

## Brand Personality / Tone
- **Serious, technical, builder-credible** — not a "look how fun SaaS can be" vibe
- **Teal/emerald accent** (`--color-accent-*`) against neutrals — distinctive but restrained
- **DM Sans throughout** — geometric, modern, confident
- **Clean minimalism with intentional density** — SEO pros expect data and specifics; avoid whitespace-for-whitespace's-sake
- **No AI slop aesthetics** — no glassmorphism, no gradient text, no "hero metrics" (1000+ customers trust us!), no generic pastel gradients

## Constraints
- Next.js 15 App Router, React 19, Tailwind v4 (CSS-first, no config file)
- No `@tailwindcss/typography` — prose is manual in `globals.css` `@layer components`
- Must work seamlessly with Vercel rewrites (`/app/*` + `/api/*` proxy to Replit PHP)
- Fast page loads — marketing site must feel snappier than the tool itself

## Differentiation
The teal rounded-lg "G" mark + bold sans wordmark is the visual anchor. Everything else supports clarity and credibility for a technical buyer deciding whether the tool is worth their time.
